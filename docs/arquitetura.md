# Arquitetura do Projeto

## Framework

- Next.js

## Linguagem

- TypeScript

## Estilização

## Organização

```

app/

components/

public/

docs/

```

Cada seção da página foi implementada como um componente independente.

Objetivo:

- organização;
- reutilização;
- manutenção simples;
- escalabilidade.
