# Decisões de Arquitetura

## Objetivo

Criar um portfólio moderno, elegante e de fácil manutenção.

## Decisões

- Next.js como framework principal.
- TypeScript para maior segurança.
- Tailwind CSS para estilização.
- Componentização por seções.
- Layout de página única (One Page).

## Filosofia

Priorizar simplicidade, organização e evolução contínua do projeto.
